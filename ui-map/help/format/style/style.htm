스타일 <F6>

자주 사용하는 글자 모양이나 문단 모양을 미리 정해 놓고 쓰는 것을 '스타일(styles)'이라고 합니다. 스타일을 만들어 놓으면 필요할 때 그 스타일을 선택하는 것만으로 해당 문단의 글자 모양과 문단 모양을 한꺼번에 바꿀 수 있습니다.

스타일은 단순히 글자 모양이나 문단 모양을 간편하게 선택하기 위해서라기보다는, 긴 글에 대하여 일관성 있는 문단 모양을 유지하면서 편집 작업을 하는 데 꼭 필요한 기능입니다.

따라 하기

[스타일] 메뉴를 이용하여 스타일 적용하기

  1. 스타일을 바꾸고 싶은 문단 위에 커서를 옮겨 놓습니다.
  2. [서식-스타일]을 실행하면 [스타일] 대화 상자가 나타납니다.
  3. [스타일 목록]에서 원하는 스타일 이름을 선택하고 [확인]을 누릅니다.
  4. 커서가 놓인 문단의 글자 모양과 문단 모양이 바뀝니다.

서식 도구 상자를 이용하여 스타일 적용하기

  1. 스타일을 바꾸고 싶은 문단 위에 커서를 옮겨 놓습니다.
  2. [서식] 도구 상자의 [스타일] 목록에서 원하는 스타일 이름을 선택합니다.
  3. 커서가 놓인 문단의 글자 모양과 문단 모양이 바뀝니다.

스타일 이름

  • 기본 스타일 이름

    한/글에서 기본으로 제공하는 스타일의 영문 이름은 다음과 같습니다. 이때, 영문 이름은 대소문자를 구별합니다.

    영역 기본 스타일 이름
    본문 Body
    개요 1 Outline 1
    ... ...
    개요 10 Outline 10
    쪽 번호 Page Number
    머리말 Header
    각주 Footnote
    미주 Endnote
    메모 Memo
    차례 제목 TOC Heading
    차례 1 TOC 1
    차례 2 TOC 2
    차례 3 TOC 3
  • 스타일 이름은 영문 이름이 우선함

    한/글에서 스타일을 구분하는 것은 스타일의 영문 이름입니다. 스타일 이름은 다르지만 영문 이름이 같은 경우에는 두 스타일을 같은 스타일로 인식합니다. 따라서 [붙이기]나 [문서 끼워 넣기]를 했을 때 같은 이름의 스타일로 처리합니다.

    따라서 기본 스타일을 편집하여 새 스타일을 만들 때에는 스타일 이름뿐만 아니라 영문 이름까지 바꾸어 주는 것이 좋습니다.

  • 영문 이름이 없는 경우

    스타일 이름이 같을 때: 같은 스타일로 처리

    스타일 이름이 다를 때: 서로 다른 스타일로 처리

  • 영문 이름을 지정한 경우

    스타일 이름과 영문 이름이 모두 같을 때: 같은 스타일로 처리

    스타일 이름은 같고 영문 이름만 다를 때: 같은 스타일로 처리

    스타일 이름은 다르고 영문 이름만 같을 때: 같은 스타일로 처리

    스타일 이름과 영문 이름이 모두 다를 때: 서로 다른 스타일로 처리

스타일 개수 제한

한 파일에 쓸 수 있는 스타일의 개수는 최대 160개입니다. 하지만 보통 한 문서에 10개 남짓의 스타일을 만들어 놓고 쓰는 것이 적당합니다.
