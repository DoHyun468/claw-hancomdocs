스타일 지우기

사용하지 않는 스타일이나 필요 없는 스타일을 지워 스타일 목록을 정리합니다.

따라 하기

  1. [서식-스타일]을 실행합니다.
  2. [스타일] 대화 상자가 나타나면 [스타일 목록]에서 지울 스타일을 선택하고 [스타일 지우기] 아이콘()을 누릅니다.
  3. 본문에서 사용되지 않는 스타일은 즉시 삭제되며, 선택한 스타일이 적용된 문단이나 글자가 있는 경우 [바꿀 스타일 선택] 대화 상자가 표시됩니다.
  4. 삭제하려는 스타일을 대신할 스타일을 선택하고 [확인]을 누릅니다.

참고

대체할 스타일

지우려는 문단 스타일이 본문에 적용되어 있으면, 지우려는 스타일이 적용된 문단에 어떤 스타일을 대신 적용할 것인가를 선택해야 합니다.

적당히 대체할 스타일이 없으면 [바탕글] 스타일을 선택합니다. 왜냐하면 사용자가 스타일을 지정하지 않은 모든 문단이 이 [바탕글] 스타일로 지정되어 있기 때문입니다.
