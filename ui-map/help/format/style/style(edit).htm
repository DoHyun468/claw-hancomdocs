스타일 편집하기

이미 있는 스타일의 이름, 문단 모양, 글자 모양, 다음 문단에 적용할 스타일 등의 스타일 내용을 고칩니다. 스타일 내용을 바꾸면 그 스타일이 적용된 모든 글자나 문단의 모양이 한꺼번에 바뀝니다.

따라 하기

  1. [서식-스타일]을 실행합니다.
  2. [스타일] 대화 상자가 나타나면 [스타일 목록]에서 편집할 스타일을 선택하고 [스타일 편집하기] 아이콘()을 누릅니다.
  3. [스타일 편집하기] 대화 상자가 나타나면 [스타일 이름]과 [다음 문단에 적용할 스타일]을 변경한 다음 [확인]을 누릅니다.
  4. [문단 모양 정보], [글자 모양 정보] 또는 [글머리표/문단 번호 정보]의 [설정] 단추를 눌러 원하는 속성을 변경합니다.
  5. 스타일 변경을 완료했으면 [스타일] 대화 상자에서 [확인] 단추를 누릅니다.

참고

스타일 편집하기

스타일 내용을 바꾸면 문서에서 그 스타일이 적용된 모든 글자나 문단의 모양이 한꺼번에 바뀝니다.

스타일을 적용한 후 사용자가 직접 글자 모양이나 문단 모양을 원래 스타일 내용과 다르게 바꾼 부분은 그대로 놓아둡니다.

스타일 내용을 바꾼 다음, 같은 문단에 다시 한 번 같은 스타일을 적용하면 "본문을 스타일 모양대로 덮어 쓸까요?" 메시지가 나타납니다. [확인] 단추를 누르면 사용자가 직접 바꾼 모양들까지 모두 새로운 스타일 내용대로 덮어 씁니다.

주의

스타일 이름은 영문 이름이 우선함

한/글에서 스타일을 구분하는 것은 스타일의 영문 이름입니다. 스타일 이름은 다르지만 영문 이름이 같은 경우에는 두 스타일을 같은 스타일로 인식합니다. 따라서 [붙이기]나 [문서 끼워 넣기]를 했을 때 같은 이름의 스타일로 처리합니다.

  • 영문 이름이 없는 경우
    • 스타일 이름이 같을 때: 같은 스타일로 처리
    • 스타일 이름이 다를 때: 서로 다른 스타일로 처리
  • 영문 이름을 지정한 경우
    • 스타일 이름과 영문 이름이 모두 같을 때: 같은 스타일로 처리
    • 스타일 이름은 같고 영문 이름만 다를 때: 같은 스타일로 처리
    • 스타일 이름은 다르고 영문 이름만 같을 때: 같은 스타일로 처리
    • 스타일 이름과 영문 이름이 모두 다를 때: 서로 다른 스타일로 처리

따라서 기본 스타일을 편집하여 새 스타일을 만들 때에는 스타일 이름뿐만 아니라 영문 이름까지 바꾸어 주는 것이 좋습니다.
