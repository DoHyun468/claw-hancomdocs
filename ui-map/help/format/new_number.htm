새 번호로 시작

쪽, 그림, 표, 수식, 각주/미주 등에 매겨진 번호를 현재 커서가 있는 쪽부터 새로운 번호로 시작하고 싶을 때 [쪽-새 번호로 시작]을 실행하여, 그 이후의 번호를 사용자가 지정한 새로운 번호로 다시 매길 수 있습니다.

따라 하기

  1. 현재 문서에서 새 번호로 시작할 쪽으로 커서를 이동합니다.
  2. [쪽-새 번호로 시작]을 실행합니다.
  3. [새 번호로 시작] 대화 상자에서 [번호 종류-쪽 번호]를 선택합니다.
  4. [시작 번호] 입력 상자에 1이라는 숫자를 입력하고 [넣기] 단추를 누릅니다.
  5. 문서의 중간쯤에 커서가 놓여 있음에도 불구하고, 현재 쪽 번호가 "1쪽"으로 시작됩니다.

자세한 정보

[새 번호로 시작] 대화 상자

번호 종류

쪽 번호, 그림 번호, 표 번호, 수식 번호, 각주 번호, 미주 번호 중에서 새 번호로 시작할 번호 종류를 선택합니다.

시작 번호

시작 번호에 대한 값을 지정합니다. 숫자로 된 쪽 번호는 0~65,535 사이에서 시작 번호 값을 지정하고 기호 또는 문자로 된 쪽 번호는 1~65,535 사이에서 시작 번호 값을 지정할 수 있습니다. 시작 번호를 새로 지정하면 시작 번호에 맞게 이후 쪽 번호가 다시 매겨집니다.

기호 또는 문자로 된 쪽 번호를 선택하고 키보드를 사용하여 시작 번호를 0으로 설정한 다음 [넣기]를 누르면 1~65,535 사이의 숫자를 누르라면 경고 메시지가 나타납니다.

참고

용도

글의 맨 앞에 놓이는 차례나 머리말이 있어도 본문을 1쪽부터 매기고 싶으면, [새 번호로 시작] 기능을 이용합니다.

주의

제한

하나의 파일에 들어갈 수 있는 쪽 번호 표시는 제한이 없으며, 같은 쪽(페이지)에 들어갈 수 있는 쪽(페이지) 번호 표시에도 제한이 없습니다. 그러나 여러 개의 쪽 번호 표시가 있을 때, 인쇄되는 내용은 각 쪽부터 맨 뒤에 있는 표시가 적용됩니다.
