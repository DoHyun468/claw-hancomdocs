변경 내용 추적

변경 내용 추적 기능을 실행하면 수정된 내용을 원본 내용과 구분하여 쉽게 확인할 수 있습니다.

따라 하기

변경 내용 추적 실행하기

  1. 편집할 문서를 불러옵니다.
  2. [검토-변경 내용 추적]을 실행합니다.
  3. 내용을 바꾸거나 추가합니다. 변경된 내용이 원본 내용과 구분되어 편집 창에 표시됩니다.

   변경 내용을 적용 또는 취소하지 않고 표시한 채로 변경 내용 추적 기능을 해제하면, 해제하기 전에 작업한 변경 내용과 원본 내용이 편집 창에 함께 보입니다. 또한 변경 추적 기능을 해제한 이후 변경한 내용은 추적할 수 없습니다.

변경 내용 적용 또는 취소하기

  1. 변경 내용 추적이 적용된 문서를 불러옵니다.
  2. 변경 내용 추적이 표시된 곳에서 마우스 오른쪽 단추를 누릅니다.
  3. 표시되는 빠른 메뉴에서 [적용] 또는 [적용 안 함] 중 하나를 누릅니다.

   [검토-변경 내용 추적-적용 후 다음으로 이동] 또는 [검토-변경 내용 추적-취소 후 다음으로 이동]을 선택하여 변경 내용을 적용하거나 취소할 수도 있습니다.

참고

변경 내용 보기

삽입하거나 삭제한 내용, 서식별로 선택하여 검토할 수 있도록 변경 내용을 표시합니다.

  • 삽입 및 삭제: 추가하고 삭제한 내용을 본문 내용과 구분해서 표시합니다.
  • 서식: 변경된 서식을 원본 서식과 구분해서 표시합니다.

최종본 및 변경 내용

변경 내용을 적용하거나 취소한 최종본 문서의 표시 방법을 선택할 수 있습니다.

  • 최종본 및 변경 내용: 수정, 추가 등 변경 내용을 반영한 최종본을 기준으로 문서에 표시합니다. 서식 변경, 추가, 삭제 등 변경 내용은 지정된 형식으로 표시합니다.
  • 최종본: 모든 변경 내용이 적용된 문서의 최종 형태를 표시합니다.
