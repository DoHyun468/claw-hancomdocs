수식 명령어 설명

수식 편집기에서 사용할 수 있는 명령어 목록과 사용 예제입니다.

자세한 정보

기본 명령어

명령어 설명 입력 예 결과
TIMES 곱셈 기호를 표시합니다. 2 times 5=10
OVER 분수를 표시합니다. 분수는 기본적으로 가운데 맞춤으로 정렬하는데, 굳이 오른쪽이나 왼쪽 맞춤을 쓰려면 <SpaceBar>를 이용합니다. 1 over 2
ATOP "OVER" 명령과 같지만 분수를 나타내는 가로선을 생략합니다. x atop y
SQRT 제곱근을 표시합니다. sqrt 2
^ 위 첨자를 표시합니다. 수식 편집 영역에서 "^"를 직접 입력하거나, 명령어 입력 창에 "SUP"를 입력합니다. E=mc^2
_ 아래 첨자를 표시합니다. 수식 편집 영역에서 "_"를 직접 입력하거나, 명령어 입력 창에 "SUB"를 입력합니다. H_2 O
INT, OINT 적분 기호를 표시합니다. int _1 ^2 {3x^2}dx
SUM 총합 기호(Sigma)를 표시합니다. sum_{x=0} ^{inf}
BIGG 기호 뒤에 쓰인 기호의 크기를 확대합니다. {a+b} over {a-b} bigg / {x+y} over {x-y}
lim, Lim 극한 기호를 표시합니다. 이때 lim과 Lim은 대소문자를 정확하게 가려 써야 합니다. y= lim _{x -> 0} {{1} over {x}}
UNION, INTER, PROD 집합 기호를 표시합니다. 명령어 앞에 "SMALL"을 덧붙이면 첨자 없는 집합 기호를 입력합니다. U=(A SMALLUNION B) SMALLINTER C
NOT 글자 앞에 not을 붙이면 그 글자에 사선을 그어줍니다. not =
REL 두 항 간의 상호 관계를 상세히 표현할 수 있게 하는 기능으로, 두 항을 연결하는 화살표의 위, 아래에 관계식 등의 내용을 삽입할 수 있습니다. A REL <-> {+2} {-5} B
BUILDREL REL과 유사한 기능으로, 화살표 아랫부분의 내용을 생략할 수 있습니다. A BUILDREL <-> {+2} B
CASES 여러 개의 행 전체를 나타내는 묶음표{를 표시합니다. 묶음표의 크기는 상황에 따라 자동으로 확대되어 표시됩니다. cases {2x+y=4 # 3x-4y=-1}
PILE, LPILE, RPILE 위에 있는 문자를 기준으로 가운데(PILE), 왼쪽(LPILE), 오른쪽(RPILE) 맞춤을 선택할 수 있습니다.    
EQALIGN 칸 맞춤 표시인 "&"를 기준으로 수식의 세로 위치를 조절해 줍니다.    
CHOOSE 또는 BINOM 조합 기호를 입력합니다. [전체항] CHOOSE [선택항] 또는 BINOM [전체항] [선택항]  
MATRIX 행렬(matrix)을 입력합니다.
matrix는 줄(행) 단위로 입력하는 일반적인 방법과 matrix 다음에 col을 입력하여 칸(열) 단위로 입력하는 방법이 있습니다.
col을 넣어 행렬을 칸 단위로 입력할 때는 각 수식 앞에 col(가운데) 대신 lcol(왼쪽 맞춤), rcol(오른쪽 맞춤)을 넣어 정렬 위치를 지정할 수도 있습니다.		 matrix{a_{1} &b_{1}&c_{1} # a_{2}&b_{2}&c_{2} # a_{3}&b_{3}&c_{3}}
PMATRIX matrix 대신에 pmatrix를 쓰면 행렬(matrix) 앞뒤로 소괄호()를 표시해줍니다. pmatrix{a_{1} &b_{1}&c_{1} # a_{2}&b_{2}&c_{2} # a_{3}&b_{3}&c_{3}}
BMATRIX matrix 대신에 bmatrix를 쓰면 행렬(matrix) 앞뒤로 대괄호[]를 표시해줍니다. bmatrix{a_{1} &b_{1}&c_{1} # a_{2}&b_{2}&c_{2} # a_{3}&b_{3}&c_{3}}
DMATRIX matrix 대신에 dmatrix를 쓰면 행렬(matrix) 앞뒤로 세로줄|을 표시해줍니다. dmatrix{a_{1} &b_{1}&c_{1} # a_{2}&b_{2}&c_{2} # a_{3}&b_{3}&c_{3}}

글자 장식 명령어

문자 또는 단어에 간단한 명령어를 앞세워 여러 가지 글자 장식을 할 수 있습니다. 해당 명령어를 먼저 입력한 후 문자는 나중에 입력합니다.

ACUTE, GRAVE, DOT, DDOT, BAR, VEC, DYAD, UNDER 등이 있습니다.

입력 결과 입력 결과
acute A bar A
grave A vec A
dot A dyad A
ddot A under A
hat A arch A
hat AA arch AA
hat AAA arch AAA
check A tilde A
check AA tilde AA
check AAA tilde AAA

입력 기호 요약

문자 또는 단어에 간단한 명령어를 넣어 수식을 편집할 수 있습니다.

~, `, { }, " ", #, & 등이 있습니다.

입력 결과
~ 빈칸 (스페이스 효과)
' 빈칸의 1/4
{   } 여러 항의 묶음
"   " 9자 이상의 한 낱말 묶음
# 줄 바꾸기 (<Enter> 효과)
& 세로 칸 맞춤 (탭 효과)

참고

사용하지 않는 명령어

Windows용 한/글 97의 [수식 편집기]에서 사용하던 다음과 같은 명령어는 더 이상 사용할 수 없습니다.

FROM, TO: 아래,위 범위

SCALE: 글자의 크기 비율(%)

PHANTOM: 허깨비

주의

대소문자 구별

대부분의 명령어는 대소문자를 구분하지 않지만 첫 글자를 대문자로 기재한 것은 반드시 대문자로 입력해야 합니다.
