책갈피 지우기

더는 필요 없는 책갈피를 지웁니다.

따라 하기

  1. [입력-책갈피]를 실행합니다.
  2. [책갈피] 대화 상자의 [책갈피 목록]에서 지우고자 하는 책갈피 이름을 선택하고 [삭제] 단추를 누릅니다.
  3. "선택한 책갈피를 지울까요?" 메시지가 나타나면 [확인] 단추를 누릅니다.
  4. 선택한 책갈피가 [책갈피 목록]에서 지워지고, 본문에 삽입된 책갈피 코드도 지워집니다.

참고

본문에서 책갈피 지우기

본문의 책갈피 조판 부호 바로 앞에 커서를 놓고 <Delete>를 누르거나 책갈피 조판 부호 바로 뒤에 커서를 놓고 <Backspace>를 누르면, 해당 책갈피가 지워집니다.
