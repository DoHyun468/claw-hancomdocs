메모

현재 입력 중인 문서에서 특정 단어나 블록으로 설정한 문자열에서 메모를 실행하면 편집 용지 기준 오른편에 글상자가 실행되며 추가 내용을 입력할 수 있습니다.

따라 하기

메모 넣기

  1. 본문 내용 중 메모를 삽입할 단어에 커서를 이동합니다.
  2. [입력-메모]를 실행합니다.
  3. 메모를 삽입한 단어를 기준으로 오른쪽에 메모 글상자가 생깁니다.

  4. 메모 글상자 안에 메모 내용을 입력한 다음 편집 창의 임의의 공간을 마우스로 눌러 메모 글상자에서 빠져나옵니다.

제한

메모 입력 상자에서의 입력 제한

메모 내용 입력을 위해 메모 입력 상자에 커서를 둔 상태에서는 [문자표]를 제외한 모든 [입력] 메뉴가 비활성화됩니다.

관련 기능

모든 메모 표시, 메모 안내선 표시
