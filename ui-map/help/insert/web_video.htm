웹 동영상

인터넷의 동영상 주소 또는 소스 코드를 사용하여 현재 문서에서 바로 재생할 수 있도록 연결합니다.

따라 하기

웹 동영상 연결하기

  1. [입력-웹 동영상]을 실행합니다.
  2. [웹 동영상 넣기] 대화 상자가 나타나면 입력란에 복사한 동영상 주소 또는 소스 코드를 붙여 넣고 [넣기] 단추를 누릅니다.
  3. 편집 창에 연결한 웹 동영상이 삽입됩니다.

자세한 정보

[웹 동영상 넣기] 대화 상자

입력란

연결할 웹 동영상 주소 또는 소스 코드를 복사하여 붙여 넣습니다.

넣기

연결할 웹 동영상을 편집 창에 삽입합니다.

취소

연결할 웹 동영상을 편집 창에 삽입하지 않고 대화 상자를 닫습니다.

연결 및 사용 가능한 입력 형식

Youtube 동영상만 연결할 수 있으며 [웹 동영상 넣기] 대화 상자의 입력란에는 다음 입력 형식을 사용할 수 있습니다.

  • www.youtube.com/watch?v="고유ID"
  • http://youtu.be/"고유ID"
  • <iframe src=“www.youtube.com/embed/"고유ID"" ...></iframe>
  • <object> ... %lt;embed src="www.youtube.com/v/"고유ID"" ...></embed></object>

주의

일부 환경에서 웹 동영상 재생 지원 안 함

웹 브라우저의 보안 수준이 [높음]으로 설정된 사용자 환경에서는 웹 동영상 재생이 지원되지 않습니다. 웹 동영상이 재생되지 않는 경우 웹 브라우저의 보안 수준을 확인하시기 바랍니다.

기업 사용자인 경우 해당 보안 수준을 변경하지 못할 수 있습니다. 자세한 내용은 조직 보안 정책 관리자에게 문의하시기 바랍니다.
