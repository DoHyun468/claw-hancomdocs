기본 도구 상자

기본 도구 상자는 문서를 작성하면서 자주 사용되는 기능을 아이콘 모양으로 만들어 모아 놓은 곳입니다.

자세한 정보

기본 도구 상자 아이콘 설명

아이콘 이름 설명
저장하기 편집 중인 파일을 저장합니다.
오려 두기 블록으로 설정된 부분을 지우면서 그 내용을 클립보드에 저장합니다.
복사하기 블록으로 설정된 부분의 내용을 클립보드에 저장합니다.
붙이기 [오려 두기] 또는 [복사하기]를 실행하여 클립보드에 저장한 내용을 커서 위치에 다시 붙여 넣습니다.
모양 복사 커서 위치의 글자 모양이나 문단 모양을 복사하여 다른 곳으로 붙여 넣습니다.
찾기 편집 중인 문서에서 특정 글자나 내용을 찾습니다.
도형 커서가 있는 곳에 도형을 삽입합니다.
그림 그림 파일을 불러와 커서가 있는 곳에 삽입합니다.
표 커서가 있는 곳에 표를 삽입합니다.
차트 커서가 있는 곳에 차트를 삽입합니다.
각주 각주 번호를 매기고 본문 아래쪽에 각주 내용을 추가합니다.
미주 미주 번호를 매기고 구역 맨 끝 또는 문서 맨 아래쪽에 미주 내용을 추가합니다.
하이퍼링크 문서의 특정한 위치에 웹 페이지나 전자 우편 주소를 연결합니다.
문자표 키보드로 입력하기 어려운 기호나 문자를 입력합니다.
글자 모양 입력할 내용이나 블록으로 선택한 글자 모양을 바꿉니다.
문단 모양 커서가 있는 내용의 문단 모양을 바꿉니다.
개체 속성 본문에 삽입한 개체의 크기, 위치, 여백 등과 같은 속성을 바꿉니다.
머리말 모든 쪽 맨 위에 표시할 내용을 지정합니다.
꼬리말 모든 쪽 맨 아래에 표시할 내용을 지정합니다.
조판 부호 문서에 조판 부호를 표시하거나 숨깁니다.
문단 부호 문서에 문단 부호를 표시하거나 숨깁니다.
격자 보기 문서에 격자를 표시하거나 숨깁니다.
