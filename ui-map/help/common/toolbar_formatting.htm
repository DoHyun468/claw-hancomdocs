서식 도구 상자

서식 도구 상자는 자주 사용하는 서식 관련 기능을, 대화 상자를 거치지 않고 한 번의 동작으로 바로 실행할 수 있도록 대화 상자의 항목을 아이콘으로 만들어 모아 놓은 곳입니다.

아이콘 이름 설명
되돌리기 문서 편집 과정에서 조금 전에 수행한 동작을 다시 원래대로 되돌립니다.
다시 실행 [되돌리기]를 실행하여 취소했던 명령이나 동작을 다시 실행합니다.
스타일 사용할 스타일을 지정합니다.
언어 [대표], [한글], [영문], [한자], [일어], [외국어], [기호], [사용자] 중에서 언어를 선택합니다. [대표] 글꼴은 언어별 글꼴에 대하여 서로 어울리는 글꼴끼리 짝지어 집합으로 만들어 놓은 것입니다.
글꼴 사용할 글꼴을 지정합니다.
글자 크기 글자 크기를 키우거나 줄일 수 있습니다. 글자의 크기를 조절하면 가로 세로가 함께 커지고 줄어듭니다.
진하게 선택한 글자를 진하게 표시합니다.
기울임 선택한 글자를 오른쪽으로 기울게 표시합니다.
밑줄 선택한 글자에 밑줄을 긋습니다. [밑줄 색]에서 밑줄에 적용할 색을 선택할 수 있습니다.
취소선 선택한 글자에 취소선을 긋습니다.
글자 색 선택한 글꼴의 글자 색을 원하는 색으로 변경합니다.
형광펜 형광펜 색을 선택하여 선택한 글자를 형광펜으로 강조 표시합니다.
양쪽 정렬 문단 모양을 양쪽으로 가지런하게 맞춥니다.
왼쪽 정렬 문단 모양을 왼쪽으로 가지런하게 맞춥니다.
가운데 정렬 문단의 글자를 가운데로 모읍니다.
오른쪽 정렬 문단 모양을 오른쪽으로 가지런하게 맞춥니다.
배분 정렬 문단 모양을 양쪽으로 가지런히 맞추되, 글자 사이를 일정하게 띄웁니다.
나눔 정렬 문단 모양을 양쪽으로 가지런하게 맞추되, 어절 사이를 일정하게 띄웁니다.
글머리표 매기기 여러 개의 항목을 나열할 때 앞부분에 원하는 모양의 글머리표를 붙입니다.
문단 번호 매기기 여러 개의 항목을 나열할 때 앞부분에 번호를 매깁니다.
한 수준 증가 개요 문단의 단계가 한 수준 씩 증가합니다.
한 수준 감소 개요 문단의 단계가 한 수준 씩 감소합니다.
줄 간격 현재 문단의 줄 간격을 바꿉니다.

참고

색상 팔레트

색상 팔레트에서는 원하는 색을 지정할 수 있습니다.

  • 테마 색

    특정 주제에 어울리는 여러 색을 조합하여 하나의 그룹으로 제공하는 색을 테마 색이라고 합니다. 오른쪽에 있는 화살표를 누르면 미리 정의된 여러 테마 색상이 목록으로 나열됩니다.

  • 스펙트럼

    그러데이션 형태의 색상 스펙트럼에서 원하는 색을 선택하거나 색상 코드를 입력하여 사용할 색을 직접 지정할 수 있습니다.
