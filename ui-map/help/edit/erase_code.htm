조판 부호 지우기

문서 전체 또는 블록으로 선택한 영역에서 특정 개체를 일괄 삭제할 수 있습니다.

따라 하기

  1. 문서의 내용 가운데 원하는 부분을 블록으로 선택합니다. 특정 부분을 선택하지 않으면 문서 전체에 적용됩니다.
  2. [편집-조판 부호 지우기]를 실행합니다.
  3. [조판 부호 지우기] 대화 상자가 나타나면 [조판 부호 목록]에서 지울 개체를 선택합니다.

    예를 들어, [각주]와 [그림]의 확인란을 선택합니다.

  4. 지울 개체를 모두 선택했으면 [지우기] 단추를 누릅니다.
  5. 설정된 영역에 입력된 모든 각주와 그림이 한꺼번에 삭제됩니다.

조판 부호 개체 목록이 나타납니다. 지우고 싶은 개체를 개체 목록에서 다중 선택한 다음 [지우기] 단추를 누르면 일괄 삭제할 수 있습니다.

  • 모두 선택: [개체 선택] 목록에 나열된 60여 개의 모든 조판 부호 개체를 선택합니다.
  • 모두 해제: [개체 선택] 목록에서 선택한 모든 조판 부호 개체를 선택 해제합니다.

참고

선택한 문서 영역에 따른 개체 삭제

조판 부호 지우기를 실행할 때 사용자가 선택한 문서 영역에 따라 삭제되는 개체가 달라집니다. 문서의 아무 곳에 커서를 놓으면 문서 전체의 개체를 대상으로 조판 부호 지우기가 실행되며, 문서의 특정 영역을 블록으로 선택하면 선택한 영역에 포함된 개체를 대상으로 이 기능이 실행됩니다.

특정 개체를 선택하면 선택한 개체에 포함된 개체만 대상으로 기능이 실행됩니다. 예를 들어 내부에 표가 포함된 표를 선택한 상태에서 조판 부호 지우기를 실행하면 선택한 표 안에 포함된 표만 삭제됩니다.
