책갈피 찾기 <Alt+G>

현재 편집하는 문서에 표시해 둔 책갈피가 있다면, 전체 책갈피 목록을 보여 주고 원하는 책갈피 위치로 쉽게 옮겨 갈 수 있습니다.

따라 하기

  1. 책갈피가 들어 있는 문서에서 [편집-찾기-찾아가기]를 실행합니다.
  2. [찾아가기] 대화 상자가 나타나면 [책갈피]를 선택합니다.
  3. 현재 문서의 전체 책갈피 목록을 보여 줍니다.
  4. 입력 상자에 찾아가고자 하는 책갈피 이름을 직접 입력하거나, 목록에서 원하는 책갈피를 선택하고 [가기] 단추를 누릅니다.
  5. 지정한 책갈피 위치로 커서가 옮겨집니다.

관련 기능

찾아가기, 쪽 찾기, 줄 찾기, 조판 부호 찾기, 책갈피
