찾아가기 <Alt+G>

줄이나 쪽과 같이 특정한 위치로 커서를 옮기거나, 그림, 표, 책갈피 등이 삽입된 위치를 찾아 커서를 옮깁니다.

한/글에서는 문단 모양이나 글자 모양 등은 해당 문단이나 글자에 속성으로 표현하여 기록하고, 그림이나 표, 머리말, 꼬리말, 책갈피 등은 커서 위치에 보이지 않는 '조판 부호'를 넣어 처리합니다. 이런 조판 부호는 [편집-찾기-찾기]로는 찾을 수 없고 [편집-찾기-찾아가기]를 실행하여 찾을 수 있습니다.

[찾아가기]에서는 '쪽, 구역, 줄, 조판 부호, 스타일, 책갈피'를 찾을 수 있습니다.

따라 하기

  1. 여러 쪽으로 된 문서에서 [편집-찾기-찾아가기]를 실행합니다.
  2. [찾아가기] 대화 상자가 나타나면 [쪽]을 선택합니다.
  3. 현재 문서의 전체 쪽 수를 계산하여 전체 쪽 번호를 보여 줍니다.
  4. 입력 상자에 "3"을 입력하고 [가기] 단추를 누릅니다.
  5. 3쪽의 맨 앞으로 커서가 옮겨집니다.

자세한 정보

[찾아가기] 대화 상자

쪽

현재 편집하는 문서의 쪽 수를 목록 상자에 보여 줍니다.

원하는 쪽 번호를 입력 상자에 입력하거나 목록에서 선택하고 [가기] 단추를 누르면 커서가 해당 위치로 옮겨집니다.

구역

현재 편집하는 문서의 구역 수를 목록 상자에 보여 줍니다.

원하는 구역 번호를 입력 상자에 입력하거나 목록에서 선택하고 [가기] 단추를 누르면 커서가 해당 위치로 옮겨집니다.

줄

현재 편집하는 문서의 줄 수를 목록 상자에 보여 줍니다.

원하는 줄 번호를 입력 상자에 입력하거나 목록에서 선택하고 [가기] 단추를 누르면 커서가 해당 위치로 옮겨집니다.

스타일

현재 편집하는 문서에 저장해 둔 전체 스타일 목록을 보여 줍니다.

원하는 스타일을 목록에서 선택하고 [가기] 단추를 누르면 해당 스타일이 적용된 위치로 커서가 옮겨집니다.

조판 부호

한/글의 모든 조판 부호를 목록 상자에 보여 줍니다.

원하는 조판 부호를 목록에서 선택하고 [가기] 단추를 누르면 해당 조판 부호가 삽입된 곳으로 커서가 옮겨집니다.

책갈피

현재 편집하는 문서에 표시해 둔 책갈피가 있다면 전체 책갈피 목록을 보여 줍니다.

원하는 책갈피를 목록에서 선택하고 [가기] 단추를 누르면 책갈피가 삽입된 곳으로 커서가 옮겨집니다.
