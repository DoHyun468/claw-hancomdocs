구역 찾기 <Alt+G>

현재 편집하는 문서 전체가 몇 개의 구역으로 나뉘어 편집되어 있다면, 전체 구역 수를 보여 주고 원하는 구역으로 쉽게 옮겨 갈 수 있습니다.

따라 하기

  1. 여러 구역으로 나뉘어 편집된 문서에서 [편집-찾기-찾아가기]를 실행합니다.
  2. [찾아가기] 대화 상자가 나타나면 [구역]을 선택합니다.
  3. 현재 문서의 전체 구역 수를 보여 줍니다.
  4. 입력 상자에 찾아가고자 하는 구역 수를 입력하고 [가기] 단추를 누릅니다.
  5. 입력한 구역의 맨 앞으로 커서가 옮겨집니다.

관련 기능

찾아가기, 쪽 찾기, 줄 찾기, 조판 부호 찾기
