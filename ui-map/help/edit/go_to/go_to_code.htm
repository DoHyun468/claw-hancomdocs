조판 부호 찾기 <Alt+G>

현재 편집하는 문서에 들어 있는 각종 조판 부호는 일반 편집 상태에서는 본문 편집 화면에서 보이지 않습니다.

그림이나 그리기 개체, 머리말, 꼬리말 등 각종 조판 부호가 실제 입력된 위치는 [찾아가기]를 실행하여 원하는 조판 부호를 선택하면 쉽게 찾을 수 있습니다.

따라 하기

  1. [편집-찾기-찾아가기]를 실행합니다.
  2. [찾아가기] 대화 상자가 나타나면 [조판 부호]를 선택합니다.
  3. 목록 상자에 한/글의 모든 조판 부호 목록이 나타납니다.
  4. 입력 상자에 찾아가고자 하는 조판 부호를 직접 입력하거나, 목록에서 원하는 조판 부호를 선택하고 [가기] 단추를 누릅니다.
  5. 지정한 조판 부호로 커서가 옮겨집니다.

참고

조판 부호 보기

일반적인 본문 편집 상태에서는 조판 부호의 위치가 보이지 않고 커서를 움직여도 조판 부호가 있는 곳을 인식하지 않으므로 찾을 수 없습니다.

만일 조판 부호 감추기 상태에서 [찾아가기]로 특정 조판 부호를 찾았을 때에는 찾은 순간에는 커서 위치에 있는 조판 부호의 이름을 상황 선에 보여 주지만, 커서를 움직이면 그 조판 부호를 다시 찾을 수 없습니다.

조판 부호가 있는 곳을 편집 화면에서 확인하려면 [보기-표시/숨기기-조판 부호]를 선택해야 합니다. [조판 부호]를 선택하면 메뉴 앞에 선택 표시()가 나타나면서 조판 부호 보이기 상태가 됩니다.
