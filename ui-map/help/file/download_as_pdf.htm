PDF로 다운로드

현재 편집 중인 문서를 다른 사람에게 공유하거나 인쇄할 수 있도록 PDF 파일로 변환하여 내 컴퓨터에 다운로드합니다.
별도의 프로그램이나 추가 기능 없이도 편리하게 PDF 파일로 다운로드할 수 있습니다.

PDF로 다운로드

편집 중인 문서를 PDF 파일로 다운로드하려면 다음을 실행합니다.

  1. 파일-PDF로 다운로드를 누릅니다. 편집 중인 파일이 PDF 파일로 변환되어 다운로드됩니다.
  2. 화면 하단에서 다운로드된 파일을 누르면 파일이 열립니다.

참고

일반적으로 변환된 PDF 파일은 C:\Users\Downloads에 다운로드됩니다.

PDF 파일이 다운로드된 폴더 위치를 확인하려면 화면 하단에 다운로드된 파일을 마우스 오른쪽 단추로 누른 다음 폴더 열기를 누릅니다.
