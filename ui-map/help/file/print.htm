인쇄 <Ctrl+P>

현재 편집 화면에 있는 문서를 프린터로 인쇄합니다.

문서 전체나 일부를 여러 매 인쇄할 수 있고, 인쇄물의 용도에 따른 확대/축소 등 다양한 인쇄 조건을 지정할 수 있습니다.

문서 인쇄

편집 중인 문서를 인쇄하려면 다음을 실행합니다.

  1. 파일-인쇄를 누릅니다.
  2. 사용자 장치에 따라 연결된 프린터 대화 상자에서 필요한 인쇄 옵션을 설정합니다.
    미리 보기 화면을 제공하는 경우 선택한 옵션이 적용된 문서를 미리 확인할 수 있습니다.
  3. 인쇄를 실행하여 문서를 출력하거나 PDF 파일로 저장합니다.
