참고

다른 이름으로 저장

관리자가 구성한 서버 설정에 따라 다른 이름으로 저장 메뉴가 나타날 수 있습니다. 저장 방식은 서버의 설정을 따르며 사용자 임의로 변경할 수 없습니다.

자동 저장 환경일 때 저장 메뉴 지원하지 않을 수 있음

자동 저장 환경에서는 문서를 저장하지 않아도 문서가 자동으로 저장됩니다. 따라서 자동 저장 환경일 때는 화면 상단 또는 전체 메뉴 목록에 저장 메뉴가 나타나지 않을 수 있습니다. 저장 방식은 서버의 설정을 따르며 사용자 임의로 변경할 수 없습니다.
