다운로드

현재 편집 중인 문서를 내 컴퓨터로 다운로드하여 인터넷이 연결되지 않은 경우에도 내 컴퓨터에서 문서를 확인하고 수정할 수 있습니다.

다운로드

편집 중인 문서를 내 컴퓨터로 다운로드하려면 다음을 실행합니다.

  1. 파일-다운로드를 누릅니다.
  2. 화면 하단에서 다운로드된 문서를 누르면 문서가 열립니다.

참고

다운로드되는 파일의 이름과 확장자는 원본 파일을 그대로 따릅니다.

일반적으로 문서는 C:\Users\Downloads에 다운로드됩니다.

문서가 다운로드된 폴더 위치를 확인하려면 화면 하단에 다운로드된 문서를 마우스 오른쪽 단추로 누른 다음 폴더 열기를 누릅니다.
