문서 정보 확인

편집 중인 문서 정보를 확인하려면 다음을 실행합니다.

  1. 파일-문서 정보를 누릅니다.
  2. 문서 정보 대화 상자가 나타나면 파일명과 마지막 저장한 날짜를 확인합니다.
  3. 문서 정보 대화 상자의 오른쪽 상단의 닫기 아이콘을 눌러 대화 상자를 닫습니다.
