빠른 교정

[빠른 교정]은 사용자가 문서 작성을 하는 도중에 잘못 입력한 단어나 오타, 띄어쓰기가 있으면 한/글에서 자동으로 틀린 낱말을 고쳐 주는 기능입니다. 현재 사용할 수 있는 빠른 교정 기능은 ["곧은 따옴표"를 “둥근 따옴표”로 자동 바꾸기]이며, 이후 더 많은 빠른 교정 항목을 제공할 예정입니다.

참고

빠른 교정 동작 시점

[빠른 교정]은 문서를 작성하는 도중에 틀린 문자열을 입력하고 <Space>나 <Tab>, <Enter> 등을 눌러 낱말 입력을 끝냈을 때에만 동작합니다.
