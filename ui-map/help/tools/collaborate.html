채팅

라이브러리에 문서를 저장하면 한컴오피스 Web을 사용하여 웹 브라우저에서 문서를 열고 공동 작업을 수행할 수 있습니다.

참여자가 변경한 내용은 실시간으로 반영되며 다른 사람이 문서를 수정하는 동안 기다리지 않아도 됩니다. 하나의 문서를 여러 명의 참여자가 확인하고 업데이트하여 공동 작업의 결과물을 만들어낼 수 있습니다. 또한 실시간 채팅을 통해 모든 참여자가 소통함으로써 보다 편리하고 효율적으로 공유 문서를 편집할 수 있습니다.

따라하기

공동 편집

다른 참여자와 문서를 공유하고 함께 편집할 수 있습니다.

  1. 공동으로 편집할 문서를 다른 참여자와 공유합니다.
  2. 브라우저 오른쪽 상단에서 편집 단추를 눌러 공동 편집에 참여 중인 사용자 이름을 확인합니다.

  3. 공통 편집을 수행합니다. 다른 사용자의 편집 위치와 변경 내용을 즉시 확인할 수 있으며 변경된 문서의 내용은 자동으로 저장됩니다.

실시간 채팅

문서를 공동으로 편집하는 동안 채팅을 통해 모든 참여자와 실시간으로 소통할 수 있습니다.

  1. 다음 중 하나를 수행합니다.
    • 공유 문서 화면 오른쪽 위에 있는 채팅 아이콘을 누릅니다.
    • 도구-채팅을 누릅니다.
  2. 메시지 입력란에 내용을 입력한 다음 <Enter> 또는 전송 단추를 누릅니다.
  3. 채팅 창에서 참여자에게 공지 사항을 전달하거나 공동 편집에 필요한 대화를 주고 받습니다.

   사용자 채팅 창의 모든 메시지를 삭제하려면 채팅 창 아래쪽에서 나에게서만 채팅 내용 모두 지우기 아이콘을 누릅니다.

   문서를 편집하는 데 방해가 되지 않도록 채팅 창 아래쪽에 있는 투명도 슬라이더를 사용하여 채팅 창의 투명도를 조절할 수 있습니다.

채팅 창에 현재 위치 공유

공동 편집자와 채팅을 하는 동안 내가 편집 중인 위치를 정확하게 공유할 수 있습니다.

  1. 다른 사람에게 공유할 위치에 마우스 포인터를 놓거나 공유할 개체를 선택한 다음 마우스 오른쪽 단추를 누릅니다.
  2. 메뉴 목록에서 채팅 창에 현재 위치 공유를 누릅니다. 채팅 창의 입력란에 현재의 위치에 대한 고유한 값이 입력됩니다.

  3. 전송 단추를 누릅니다. 채팅 창의 대화 영역에 공유된 위치로 이동할 수 있는 링크가 전달되며 채팅 참여자라면 누구든지 링크를 눌러 해당 위치로 이동할 수 있습니다.

자세한 정보

공동 편집 시 사용자 구분

  • 한워드

    다른 사람이 편집 중인 문단이나 단어에 사용자 색상의 커서가 표시되며, 이곳에 마우스 포인터를 올려 놓으면 사용자 이름이 표시됩니다.

  • 한셀

    다른 사람이 편집 중인 셀에 사용자 색상의 테두리가 표시되며, 이곳에 마우스 포인터를 올려 놓으면 사용자 이름이 표시됩니다.

  • 한쇼

    여러 사용자가 동시에 하나의 슬라이드를 편집하면 슬라이드 썸네일 영역 사용자 아이콘이 표시되며, 이곳에 마우스 포인터를 올려 놓으면 사용자 이름이 표시됩니다.

    다른 사람이 편집 중인 글상자에 사용자 색상의 테두리가 표시되며, 이곳에 마우스 포인터를 올려 놓으면 사용자 이름이 표시됩니다.

  • 한글

    다른 사람이 편집 중인 부분에 사용자 색상의 커서가 표시되며, 현재 편집자가 편집을 완료할 때까지 다른 편집자가 내용을 변경할 수 없도록 해당 문단이나 문장이 잠깁니다. 보기-작업 창을 누르면 현재 협업에 참여 중인 공통 편집자를 확인할 수 있습니다.

제한

한워드

협업에 참여 중인 여러 사람이 동시에 입력 취소를 시도하면 되돌리기 메뉴가 비활성화될 수 있습니다. 협업하는 동안에는 셀 병합 메뉴를 사용할 수 없습니다.

한셀

문자 단위가 아닌 셀 단위의 협업을 지원합니다. 공동 편집자 중 한 명이 스프레드시트를 삭제하면 다른 편집자에게 알림 메시지가 표시되며 편집이 제한됩니다.

한쇼

다른 사람이 편집 중인 글상자에 사용자 이름과 테두리만 표시되며, 글상자 안에서의 편집 위치는 커서로 표시되지 않습니다. 협업에 참여 중인 여러 사람이 동시에 입력 취소를 시도하면 되돌리기 메뉴가 비활성화될 수도 있습니다.

한글

다른 사람이 편집 중인 문장이나 셀(표 안의 특정 셀)에 잠금 아이콘이 표시되며 공동 편집자의 편집이 제한됩니다.
