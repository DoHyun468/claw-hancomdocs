접근성 설정

'웹 접근성'이란 장애인이나 고령자는 물론 어떤 사용자도 웹 사이트에서 제공하는 정보를 비장애인과 동등하게 접근하고 활용할 수 있도록 보장하는 것입니다. (주)한글과컴퓨터도 모든 사용자가 문서 내용을 읽고 탐색할 수 있도록 웹 접근성 준수에 최선을 다하고 있습니다.

한글에서는 스크린 리더가 문서에 접근하여 문서의 내용을 소리내어 읽어줄 수 있도록 설정할 수 있습니다. 웹 접근성 향상을 위해 앞으로 더 많은 도구와 기능이 지원될 예정입니다.

스크린 리더 지원 사용 설정

스크린 리더가 문서에 접근하여 문서의 내용을 소리내어 읽어줄 수 있도록 설정할 수 있습니다.

  1. 스크린 리더 지원 사용을 설정하려면 다음 중 하나를 수행합니다.
    • 도구-접근성 설정-스크린 리더 지원 사용을 누릅니다.
    • <Ctrl+Alt+F1>을 누릅니다.
  2. 스크린 리더 지원 사용을 설정 해제하려면 다음 중 하나를 수행합니다.
    • 도구-접근성 설정-스크린 리더 지원 사용을 다시 누릅니다.
    • <Ctrl+Alt+F1>을 다시 누릅니다.

참고

문서의 접근성을 높일 수 있는 방법

신체적 또는 인지적 능력이 다른 사용자가 용이하게 접근할 수 있는 문서를 만들기 위한 몇 가지 방법을 소개합니다.

  • 색에 무관하게 내용을 인식할 수 있도록 작성
    시력이 낮거나 시각 장애가 있거나 색맹인 사용자는 색으로 구분된 내용의 의미를 놓칠 수 있습니다. 예를 들어 경고하는 내용을 빨간색으로만 강조하는 대신 '경고'라는 글자를 추가해야 색에 관계 없이 사용자가 이해할 수 있습니다.
  • 가독성 고려
    시각 장애를 가진 사용자나 고령자도 콘텐츠를 인식할 수 있도록 문서 내용과 배경 간의 명도 대비가 충분하도록 문서를 작성합니다. 기울임이나 밑줄과 같은 서식을 과도하게 적용하지 않고, 자간이나 공백을 충분히 포함합니다.
  • 스크린 리더 사용자가 문서를 쉽게 탐색할 수 있도록 내용 작성
    한글에서 기본적으로 제공되는 제목과 스타일을 사용하여 내용을 구조화하고, 맥락을 쉽게 이해할 수 있도록 논리적인 순서로 내용을 구성합니다.
  • 하이퍼링크 표시 텍스트 및 화면 설명 추가
    스크린 리더 사용자가 하이퍼링크나 시각적 콘텐츠에 대한 자세한 정보를 음성을 통해 인식할 수 있도록 대체 텍스트를 추가합니다. 스크린 리더는 화면을 볼 수 없는 사용자를 위해 추가된 대체 텍스트를 읽어줍니다.
