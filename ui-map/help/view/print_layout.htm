쪽 윤곽 <Ctrl+G+L>

한글에서 작성한 문서는 실제로 인쇄해야 비로소 각 쪽의 윤곽을 정확히 알 수 있습니다.

하지만 [쪽 윤곽]을 사용하면 문서를 인쇄하기 전 용지 여백, 머리말/꼬리말, 테두리 등이 어떻게 인쇄될지 미리 확인하면서 수정할 수 있습니다.

따라 하기

쪽 윤곽 보이기/숨기기

  1. [보기-쪽 윤곽]을 실행합니다.
  2. [쪽 윤곽] 메뉴 앞에 선택 표시가 사라지고 쪽 윤곽 보기 상태가 해제됩니다.

  3. [보기-쪽 윤곽]을 다시 실행합니다.
  4. [쪽 윤곽] 메뉴가 선택되면서 쪽 윤곽 보기 상태가 됩니다.

참고

자동으로 쪽 윤곽 켜짐

쪽 윤곽 보기를 선택하지 않고 [머리말/꼬리말]을 만들거나 [보기-쪽 모양]에서 [두 쪽]을 실행할 경우 [보기-쪽 윤곽]이 자동으로 켜져 쪽 윤곽 보기 상태가 됩니다.

위와 같은 상태에서 머리말/꼬리말 편집을 끝내거나 화면 확대 비율을 바꾸더라도 쪽 윤곽 보기 상태는 계속 유지되며, 쪽 윤곽 꺼짐 상태로 되돌아가지 않습니다.

관련 기능

확대/축소, 머리말/꼬리말
