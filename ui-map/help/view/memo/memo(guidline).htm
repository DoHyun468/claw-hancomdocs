메모 안내선 표시

[보기-메모-메모 안내선 표시]는 메모와 메모 글상자를 이어주는 안내선을 보이거나 감출 때 사용하는 전환 메뉴입니다. [메모 안내선 표시] 앞에 선택 표시가 있으면 메모 안내선 보이기 상태이고, [메모 안내선 표시] 앞에 선택 표시가 없으면 메모 안내선 감추기 상태입니다.

따라 하기

메모 안내선 보이기/숨기기

  1. 메모가 입력된 문서를 불러옵니다.
  2. [보기-메모-메모 안내선 표시]를 실행합니다.
  3. [메모 안내선 표시] 메뉴 앞에 선택 표시()가 사라지면서 문서에 삽입된 메모의 안내선이 감춰집니다.

  4. 다시 [보기-메모-메모 안내선 표시]를 실행합니다.
  5. [메모 안내선 표시] 메뉴 앞에 선택 표시가 나타나면서 문서에 삽입된 메모의 안내선이 나타납니다.
