모든 메모 표시

[모든 메모 표시]는 편집 문서에 삽입된 메모를 표시하거나 감추는 기능으로, 메모가 삽입된 문서에서 메모 내용을 보고 싶지 않을 때 사용하면 편리합니다.

따라 하기

모든 메모 표시

  1. 메모가 입력된 문서를 불러옵니다.
  2. [보기-메모-모든 메모 표시]를 실행합니다.
  3. [모든 메모 표시] 메뉴 앞에 선택 표시()가 사라지면서 문서에 입력된 전체 메모가 감춰집니다.
  4. 다시 [보기-메모-모든 메모 표시]를 실행합니다.
  5. [모든 메모 표시] 메뉴 앞에 선택 표시가 나타나면서 문서에 입력된 전체 메모가 표시됩니다.

참고

모든 메모 표시

[보기-메모-모든 메모 표시]는 메모 보이기와 메모 감추기가 번갈아 선택되는 전환 메뉴입니다. [모든 메모 표시] 앞에 선택 표시()가 있으면 메모 보이기 상태이고, [모든 메모 표시] 앞에 선택 표시가 없으면 메모 감추기 상태입니다.
