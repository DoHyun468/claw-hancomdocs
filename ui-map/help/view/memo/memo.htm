메모

현재 입력 중인 문서에서 특정 단어나 블록으로 설정한 문자열에서 메모를 실행하면 편집 용지 기준 오른편에 글상자가 실행되며 추가 내용을 입력할 수 있습니다.

참고

모든 메모 표시

편집 문서에 삽입된 메모를 표시하거나 감추는 기능으로, 메모가 삽입된 문서에서 메모 내용을 보고 싶지 않을 때 사용하면 편리합니다.

[보기-메모-모든 메모 표시]는 메모 보이기와 메모 감추기가 번갈아 선택되는 전환 메뉴로, [모든 메모 표시] 앞에 선택 표시()가 있으면 메모 보이기 상태이고, [모든 메모 표시] 앞에 선택 표시가 없으면 메모 감추기 상태 표시입니다.

메모 안내선 표시

메모와 메모 글상자를 이어주는 안내선을 보이거나 감출 때 사용합니다.

[보기-메모-메모 안내선 표시]는 메모와 메모 글상자를 이어주는 안내선을 보이거나 감출 때 사용하는 전환 메뉴입니다. [메모 안내선 표시] 앞에 선택 표시()가 있으면 메모 안내선 보이기 상태이고, [메모 안내선 표시] 앞에 선택 표시가 없으면 메모 안내선 감추기 상태 표시입니다.

조판 부호가 켜진 상태

조판 부호를 켜면 편집 화면에 [메모 시작] [메모 끝] 필드가 나타나고, 메모 글상자는 화면에 나타나지 않습니다.
