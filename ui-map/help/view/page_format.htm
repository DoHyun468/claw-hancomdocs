쪽 모양

화면에 표시할 쪽 개수를 선택합니다.

따라 하기

  1. [보기-쪽 모양]을 실행합니다.
  2. 원하는 쪽 모양을 선택합니다.

참고

쪽 모양

  • 한 쪽

    편집 용지의 크기와 관계없이 현재 문서의 한 쪽 분량을 한 화면에 표시합니다.

  • 두 쪽

    편집 용지의 크기와 관계없이 현재 문서의 두 쪽 분량을 나란히 볼 수 있도록 한 화면에 표시합니다.

    이때 화면에 나타나는 '두 쪽'의 기준은 쪽 번호와 관계없으며, 처음부터 두 쪽씩 순서대로 보여 줍니다.
