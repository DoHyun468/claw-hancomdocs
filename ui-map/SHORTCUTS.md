# 한컴독스(webhwp) 단축키 일람

> **출처**: 공식 도움말 `shortcut/shortcut.htm` (https://webhwp.hancomdocs.com/cloud-hwp/help/Hwp/ko_kr/index.htm#t=shortcut%2Fshortcut.htm), 2026-06-11 수집.
> **⚠️ 자동화 관점 주의** (MENU_MAP.md 원칙과 동일): webhwp 단축키는 브라우저 단축키와 충돌 시 **브라우저가 우선**하고(예: Chrome Ctrl+N), 기기/브라우저별로 동작이 다를 수 있어 **신뢰 불가 → 클릭(셀렉터)이 정석, 단축키는 참고/보조**. 단, 표 내부 셀 조작(F5 계열)·커서 이동·블록 설정처럼 메뉴에 없는 키들은 실제 op에서 쓰는 핵심 수단.
> 3키 조합 `Ctrl+글쇠1,글쇠2` = Ctrl+글쇠1 누르고 뗀 후 글쇠2.
> macOS에서 기능 키(F7 등)가 안 들으면 `Fn+F7`처럼 Fn 병행 (macOS 설정 의존).
