# 특수문자 입력 레퍼런스

**핵심: 특수문자는 별도 다이얼로그 없이 텍스트 op의 `--text`(또는 `--to`, `--find` 등)에 그대로 넣으면 된다.**
`insert-text` · `replace-text` · `set-cell-text` · `pinpoint --insert/--replace` · `textbox --text` · `footnote/endnote/memo --text` 전부 동일.

```bash
node hancom.js insert-text --name <문서이름> --anchor "<기준>" --text "온도는 25℃, 면적 ㎡당 ₩1,000 → 자세한 건 ① 참고 ※" --apply
```

아래 문자들은 실제 입력→저장→렌더까지 확인된 목록(자주 쓰는 것 위주). 여기 없는 유니코드 문자도 대부분 그대로 들어간다.

## 이모지
`🎉` `✅` `😀` 같은 컬러 이모지도 입력·저장·렌더 확인됨. 단 **한컴독스 웹 에디터 기준** — 다른 HWP 뷰어/프로그램에서 똑같이 보일지는 보장 못 하니, 격식 문서엔 위 흑백 기호 계열이 안전.

## 주의
- 아주 희귀한 문자는 글꼴에 글리프가 없어 `□`로 보일 수 있다 — 넣은 뒤 `around --text "<그 문자 앞뒤>" --zoom`으로 캡처해 확인 권장.
